# OpenStore

Cliente alternativo do F-Droid para Android.

## Recursos
- Sincroniza repositórios F-Droid via `index-v1.jar` com validação de assinatura (fingerprint SHA-256).
- Lista apps com busca.
- Tela de detalhes com download de APK e verificação SHA-256 antes de instalar.

## Build
Abra no Android Studio e rode o módulo `app`.
